# Animal Shelter Management System

This project is a Python-based Animal Shelter Management System developed as part of the Class 12 Computer Science curriculum.
The application provides a graphical user interface for managing the activities of an animal shelter, including animal registration, adoption management, and viewing shelter data.

## Features
- User registration and login system
- Add animals to the shelter database
- Animal adoption tracking
- Shelter data viewing
- Global data and statistics
- Graphical data visualization using Matplotlib

## Technologies Used
- Python
- Tkinter (GUI)
- MySQL Database
- NumPy
- Matplotlib
